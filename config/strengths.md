# Model Strengths Guide (read by Supervisor)

**grok-3 (xAI)**: Strong at critical thinking, spotting flaws, creative alternatives, and explaining reasoning clearly. Good devil's advocate and for tool-heavy tasks.

**claude-4-sonnet (Anthropic)**: Highest code quality, follows complex instructions reliably, excellent at secure and maintainable implementations. Prefers careful, step-by-step work.

**gemini-2.5-pro (Google)**: Fast at synthesizing information from multiple sources, handles very large context well, good at producing clean merged outputs quickly.

Use this when deciding which model is best suited for a subtask or for the final synthesis step.
