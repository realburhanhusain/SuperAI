How These Were Planned to Be Used in SuperAI

Core Architecture: SuperAI would act as the supervisor layer on top of these systems.
Memory: Mempalace-inspired structure inside MemoryPalace.
Agent Patterns: LLM_Council + Hermes-Agent + OpenClaw patterns for deliberation, delegation, and self-improvement.
External CLIs: Treat Claude Code, Aider, Cursor, etc., as first-class workers with seamless delegation.
Skills: Combine skills system with patterns from OpenClaw and Hermes-Agent.
